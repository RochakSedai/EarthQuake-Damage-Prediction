About Earthquake Building Damage Prediction

The project "Earthquake Damage Prediction System" is a Machine learning project developed using Azure technologies like ML studio for Microsoft Virtual Hackathon 2022. This system is developed with aim to predict damage of buildings due to eathquake. Currently, our models have been trained and tested using only Nepali earthquake dataset taken from Kaggle.com Dataset . Our system can currently predict the earthquake damage grades on buildings with test f1-score of 88%. We have developed this system with two modes of input i-e normal user mode and professional user mode. Here a normal user will enter his/her building id and then get the damage grade predicted for that id. Similarly, the professional user will have to enter some more details of the building like number of floors, height of building, age etc and then get the damage grade predicted for that building.
P.S: Our system is not perfect and this is just a simple prototype for the idea we presented in this hackathon
